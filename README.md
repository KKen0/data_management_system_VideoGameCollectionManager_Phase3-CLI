# data_management_system_VideoGameCollectionManager_Phase3-GUI

A Java desktop application for organizing and maintaining a personal video game library. The application uses a gamer-themed dark mode interface and allows users to load a text file, manage game records, search and sort their collection, and export a backlog report.

## Overview

The Video Game Manager was developed as a course project to demonstrate object-oriented programming, file handling, and graphical user interface design in Java. The application separates the project into three main parts:

- `Game.java` for the game data model
- `GameManager.java` for file management and collection logic
- `VGMAppr.java` for the graphical user interface

This project is designed for users who want a simple way to track their games, progress, and backlog.

## Features

- Gamer-style dark mode user interface
- Load an existing `.txt` data file
- Open and use a new or empty text file
- Display the selected file name in the application
- Add new games
- Update existing games
- Delete one or more games
- Search games using a search bar
- Sort table columns by clicking the column headers
- Show completion status as text
- Export a backlog report in `.csv` or `.txt` format
- Disable action buttons until a file is selected

## Technologies Used

- Java
- Java Swing
- Object-Oriented Programming
- File I/O
- JTable
- TableRowSorter
